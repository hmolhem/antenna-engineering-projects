## Lifecycle

Each major repository milestone receives a numbered handoff when the implementation package is prepared. After merge, the same handoff is closed out by recording the pull-request number, final status, and merge commit SHA. Minor documentation corrections do not require a separate numbered handoff unless they establish a new milestone or materially change the engineering package.
