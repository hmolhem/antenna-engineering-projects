\section{Engineering Scope and Deliverables}
\label{sec:engineering-scope}

This project develops a complete antenna-engineering workflow consisting of three connected
technical stages: design and tuning of a single microstrip patch antenna, extension of the tuned
element into a two-port \(2\times1\) patch array, and demonstration of phase-controlled beam
steering.

The analytical and simulation workflow includes the following tasks:
\begin{itemize}
    \item determine the operating frequency and substrate parameters;
    \item calculate the initial rectangular-patch and inset-feed dimensions;
    \item construct a parameterized single-patch CST model;
    \item tune the resonant frequency and input impedance;
    \item evaluate reflection coefficient, radiation pattern, directivity, radiation efficiency,
    and total efficiency;
    \item compare PEC and finite-conductivity metallization models;
    \item construct and independently retune a two-port patch array;
    \item quantify input matching and mutual coupling;
    \item combine the individual port farfields using controlled amplitude-phase excitation; and
    \item evaluate broadside and steered radiation patterns.
\end{itemize}

The report emphasizes traceability between analytical calculations, CST model parameters,
simulation results, and engineering interpretation. Key dimensions, material assumptions, solver
settings, numerical limitations, and validation evidence are documented explicitly so that the
design process can be independently reviewed and reproduced.
