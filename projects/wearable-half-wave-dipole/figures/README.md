# Curated Figures

This directory contains a compact subset of CST exports for rapid GitHub review. The complete figure set used by the report is preserved under `source/figures/`.

- `free-space/`: tuned geometry and free-space return loss
- `body-loading/`: S11 and 3D gain comparisons for 10 mm, 5 mm, and 0.1 mm clearances
- `retuning/`: selected body-loaded retuning results
